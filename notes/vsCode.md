move lines up/down
option + arrow (+shift to copy)

delete line
shift cmd k

insert line above/below
(shift) cmd enter

jump to matching bracket
shift cmd \

indent/outdent line
cmd [ or ]

fold region
opt cmd ][

fold all subregions
cmd k then cmd ][

fold ALL regions
cmd k then cmd o/j

add/remove line comment
cmd k then cmd c/u

toggle block comment
shift opt A

insert cursor
opt click, opt cmd arrow, (cmd U to undo last)
